    Forecastly

    Version 1.0.0

    App Description

    Forecastly provides accurate, real-time weather information for locations worldwide. Get detailed forecasts, severe weather alerts, and beautiful visualizations to plan your day perfectly.

    Key Features
    • Real-time weather updates
    • 5-day forecasts
    • Multiple location support
    • Interactive weather maps
    • Customizable notifications
    Technical Info
    • Powered by OpenWeatherMap API
    • Built with Bootstrap 5
    • 100% client-side JavaScript
    • Responsive design
    User Guide

    1. Type your city name in the search bar
    2. Press Enter or click the Search button
    3. View current weather conditions
    4. Scroll down for extended forecasts

    Temperature

    Shows current, feels-like, min and max temperatures in Celsius

    Humidity

    Percentage of water vapor in the air

    Wind

    Speed in m/s and direction in degrees

    Sunrise/Sunset

    Local times for daylight changes

    No Results Found
    • Check your internet connection
    • Verify city name spelling
    • Try broader location (e.g., state instead of city)
    Data Not Updating
    • Refresh the page
    • Wait 1-2 minutes for API limits
